Internal Page Navigation

An internal page navigation for long content pages using anchor tags.

To use a sub navigation for long content pages, add the with-subnav class to the <body> or <div> element. Any <h2> element inside the <body> or <div> element will be used by the JavaScript to create the sub menu box, which will be fixed to the top of the page. The JavaScript will automatically anchor each menu item to the contents.

SEE DEMO

NOTE: The <h2> elements are required for JavaScript. They will be used to create the sub menu navigation.

Code

<body class="with-subnav">
		
	<h2>...</h2>
	<p>...</p>
	
	<h2>...</h2>
	<p>...</p>
	
	<h2>...</h2>
	<p>...</p>
	
</body>

<!-- can also be done by using a div element -->

<div class="with-subnav">

	<h2>...</h2>
	<p>...</p>
	
	<h2>...</h2>
	<p>...</p>
	
	<h2>...</h2>
	<p>...</p>
	
</div>
