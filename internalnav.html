Internal Page Navigation

An internal page navigation for long content pages using anchor tags.

To use a sub navigation for long content pages, add the with-subnav class to the <div> element. Any <h2> element inside the <div> element will be used by the JavaScript to create the sub menu box. The JavaScript will automatically anchor each menu item to the contents.

D2L v10.1 Changes

Due to the new structure in D2l v10.1, the internal page navigation menu area is no longer fixed to the top (or anywhere) of the page. Fixing the menu area on top (or anywhere) of the page will cause D2L to go banana. D2L will continuously increasing the height of the iFrame, which makes the page longer and longer every second. This issue is D2L's responsibility to resolve it.

SEE DEMO

NOTE: The <h2> elements are required for JavaScript. They will be used to create the sub menu navigation.

Code

<div class="with-subnav">

	<h2>...</h2>
	<p>...</p>
	
	<h2>...</h2>
	<p>...</p>
	
	<h2>...</h2>
	<p>...</p>
	
</div>
