Helpers

Miscellaneous classes to help style the page content further.

Clearfix Class

Clear CSS float property on any elements. Usually used with columned page contents.

Code

<div class="clearfix"></div>

The push class is removed due to the changes in D2L v10.1. Existing courses use this class will not be affected by this change. The starter files has been updated to reflect this change.
