Popover

A small overlays of content that appears when the mouse click on an icon, image, or text in a page content. Do not use this component for tooltips and on hyperlinks.

Popover has four options: top, bottom, right, and left. Each option will be explained below with code and examples. To use popover, add the with-popover class and the data-title attribute to the HTML elements including <span>, <img>, etc. The data-title attribute is used to hold the popover content. Thus, it is required.

NOTE: The <a> and <abbr> elements do not support the with-popover class.

Use responsibility!

Use popover for non-essential information. Popover adds a nice interaction to the content but excessive use will cause unintended learning problems.

Top

By default, the popover will be displayed on the top of the element. The top class is optional.

Code

<p>Lorem ipsum dolor sit amet, consectetur elit. Vitae, <span class="with-popover" 
data-title="Doloribus is so awesome. I do not even know what it means.">doloribu
</span> aperiam temporibus iusto est vel aut aliquam ipsum quidem consectetur
possimus laboriosam iste molestiae consequatur magnam soluta porro commodi quia?</p>

Result

Lorem ipsum dolor sit amet, consectetur elit. Vitae, doloribus aperiam temporibus iusto est vel aut aliquam ipsum quidem consectetur possimus laboriosam iste molestiae consequatur magnam soluta porro commodi quia?

Bottom

The popover will be placed at the bottom of the text. To specify the bottom position, add the bottom class.

Code

<p>Lorem ipsum dolor sit amet, consectetur <span class="with-popover bottom"
data-title="Adipisicing has nice sound to it. I don't know what it means.">
adipisicing</span> elit. Vitae, doloribus aperiam temporibus iusto est vel aut
aliquam ipsum quidem consectetur possimus laboriosam iste molestiae consequatur
magnam soluta porro commodi quia?</p>

Result

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Vitae, doloribus aperiam temporibus iusto est vel aut aliquam ipsum quidem consectetur possimus laboriosam iste molestiae consequatur magnam soluta porro commodi quia?

Image

Popover can be used on images too!

Code

<img class="with-popover right" data-title="All you ever needed!"
src="src/img/id_logo.png" width="275" height="236" alt="Style Guide" />

Result

Warning!

Popover for image will not be displayed when printed. This is a limitation of CSS3. Avoid using image with popover for essential information. Use image caption instead.
