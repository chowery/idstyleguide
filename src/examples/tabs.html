Study Guide

The information in this section provides additional guidance when preparing for the assessments.

Demonstrate proficiency with HTML elements pertaining to web page structure, phrases and paragraphs, lists, tables, HTML links, and images

You should be familiar with, and be able to effectively use, the following aspects of HTML:

Webpage Structure

  • <!doctype …>
  • <html>…</html>
  • <head>…</head>
  • <body>…</body>
  • <div>… </div>
  • <span>…</span>
  • <h1>…</h1> through <h6>…</h6>
  • <hr />
  • <title>…</title>
  • Comments <!-- … -->

Phrases and Paragraphs - W3schools.com/Paragraphs

  • <p>…</p>
  • <br />
  • <sub>…</sub>
  • <sup>…</sup>
  • <pre>…</pre>
  • <blockquote>…</blockquote>

Lists - W3schools.com/Lists

  • <ol>…</ol>
  • <ul>…</ul>
  • <li>…</li>
  • <dl>…</dl>
  • <dt>…</dt>
  • <dd>…</dd>

Tables - W3schools.com/Tables

  • <table>…</table>
  • <tr>…</tr>
  • <td>…</td>
  • <th>…</th>
  • Row spans
  • Column spans

HTML Links - W3schools.com/Links

  • Types of links
    • Externals links
    • Internal Links (anchors)
  • Addressing
    • Absolute addressing (<a href=”http://www...>)
    • Relative addressing (<a href=”../folders/…>)

Images - W3schools.com/Images

  • Image formats (.bmp, .jpg, .png, .gif, .tiff)
  • Image manipulation
    • Change image format
    • Resizing images
    • Image cropping
    • Finding rgb values of image colors
  • Insert images in web page
    • <img src=”…” alt=”…” />
    • Link to an image
    • Use an image to link to a web page
    • Use an image to link to another image (as in thumbnails)
Demonstrate the use of the following common attributes: id, class, style and title

Attributes provide additional information about HTML elements.

  • HTML elements can have attributes
  • Attributes provide additional information about an element
  • Attributes are always specified in the start tag
  • Attributes come in name/value pairs like: name="value"

To find out more about HTML attributes use the link below. In addition to the links provided, you can search the sites listed in the General Resources tab.

W3schools: Attributes
Demonstrate the use of the following special characters: &nbsp; and &#40;

Some characters are reserved in HTML.It is not possible to use the less than (<) or greater than (>) signs in your text, because the browser will mix them with tags. To actually display reserved characters, we must use character entities in the HTML source code.

To find out more about special characters use the links below. In addition to the links provided, you can search the sites listed in the General Resources tab.

W3schools: Non-breaking space (&nbsp;) Character Reference Chart (&#40)
