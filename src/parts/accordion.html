Accordion (the GUI one)

A vertically stacked list of contents on a page without losing usability if used correctly.

To organize information with accordion, create a <div> element with the with-accordion class. Nested inside the <div> element, two <div> elements are required with accordion-title and accordion-content respectively. Repeat these two elements for another collapse panel.

Note: apply active class to the accordion-title to show on panel on page load. The use of active class is optional for accordion.

Code

<div class="with-accordion">

	<div class="accordion-title active">Item 1</div>
	<div class="accordion-content">
		<p>...</p>
	</div>
	
	<div class="accordion-title">Item 2</div>
	<div class="accordion-content">
		<p>...</p>
	</div>
	
	<div class="accordion-title">Item 3</div>
	<div class="accordion-content">
		<p>...</p>
	</div>
	
</div>
