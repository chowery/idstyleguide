Tables

The width of the <table> element will always be 100%; thus, it fills up the whole width of the page or container. The <table> is a block-level element. The use of <thead> is optional but recommended. The use of <tbody> is required in table with classes. The cells of the table will auto-resize accordingly to the content they held.

Basic Table

Code

<table>
	<thead>
		<tr>
			<th>ID</th>
			<th>First Name</th>
			<th>Last Name</th>
			<th>Username</th>
		</tr>
	</thead>
	<tbody>
		<tr>
			<td>1</td>
			<td>Ethan</td>
			<td>Lin</td>
			<td>ethan.lin</td>
		</tr>
		<tr>
			<td>2</td>
			<td>Patrick</td>
			<td>Wirth</td>
			<td>patrick.wirth</td>
		</tr>
		<tr>
			<td>3</td>
			<td>Bryan</td>
			<td>Bortz</td>
			<td>bryan.bortz</td>
		</tr>
		<tr>
			<td>4</td>
			<td>John</td>
			<td>Oppenheimer</td>
			<td>john.oppenheimer</td>
		</tr>
		<tr>
			<td>5</td>
			<td>Kristin</td>
			<td>Barnes</td>
			<td>kristin.barnes</td>
		</tr>
		<tr>
			<td>6</td>
			<td>Nate</td>
			<td>Ewings</td>
			<td>nate.ewings</td>
		</tr>
	</tbody>
</table>

Result

ID First Name Last Name Username
1 Ethan Lin ethan.lin
2 Patrick Wirth patrick.wirth
3 Bryan Bortz bryan.bortz
4 John Oppenheimer john.oppenheimer
5 Kristin Barnes kristin.barnes
6 Nate Ewings nate.ewings

Striped Table

Add "zebra-striping" to odd table rows within the <tbody> element by using the striped class.

Code

<table class="striped"></table>

Result

ID First Name Last Name Username
1 Ethan Lin ethan.lin
2 Patrick Wirth patrick.wirth

Table with Contextual Classes

Change the background of rows or cells with a meaningful color by using the following class: info (light blue), success (green), warning (yellow), and danger (red). Please use these contextual classes wisely.

Code

<table>
	<thead>
		<tr>
			<th>ID</th>
			<th>First Name</th>
			<th>Last Name</th>
			<th>Username</th>
		</tr>
	</thead>
	<tbody>
		<tr class="info">
			<td>1</td>
			<td>Ethan</td>
			<td>Lin</td>
			<td>ethan.lin</td>
		</tr>
		<tr>
			<td>2</td>
			<td>Patrick</td>
			<td class="success">Wirth</td>
			<td>patrick.wirth</td>
		</tr>
		<tr>
			<td>3</td>
			<td>Bryan</td>
			<td>Bortz</td>
			<td class="warning">bryan.bortz</td>
		</tr>
		<tr>
			<td>4</td>
			<td>John</td>
			<td class="danger">Oppenheimer</td>
			<td>john.oppenheimer</td>
		</tr>
		<tr>
			<td>5</td>
			<td>Kristin</td>
			<td>Barnes</td>
			<td>kristin.barnes</td>
		</tr>
		<tr>
			<td>6</td>
			<td>Nate</td>
			<td>Ewings</td>
			<td>nate.ewings</td>
		</tr>
	</tbody>
</table>
