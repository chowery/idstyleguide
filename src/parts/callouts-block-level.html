Block-Level Callouts

Basic Horizontal

Basic callouts use callout class with a <div> element. For callout heading use <h4> element, and <p> element for body copy. For different variations of callout, apply any of the following classes: success, info, warning, or danger.

Code

<div class="callout">
	<h4>Something!</h4>
	<p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Repellendus, recusandae
	impedit quod consectetur corporis neque suscipit laboriosam eum. Neque sit ipsa est
	consequatur illum optio reprehenderit atque deserunt facere quo.</p>
