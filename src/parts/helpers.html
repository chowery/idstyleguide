Helpers

Miscellaneous elements and classes to help style the page content further.

Page Container Class

A fluid container with left and right paddings to center a page's contents by applying page-container class to a <div> element.

Code

<div class="page-container">
...
</div>

Clearfix Class

Clear CSS float property on any elements. Usually used with columned page contents.

Code

<div class="clearfix"></div>
