Page Headers

A page must starts with a page header. Page header may not be used multiple times throughout the page.

Basic Page Header

A basic page header contains only the page title.

Code

<header>
	<h1>Page title goes here (30 pixels)</h1>
</header>

Result

Page title goes here (30 pixels)

Page Header with Subtext

A page header with subtext contains the page title and supplement text

Code

<header>
	<h1>Page title goes here (30 pixels) <small>subtext goes here (24 pixels)</small></h1>
</header>

Result

Page title goes here (30 pixels) subtext goes here (24 pixels)
