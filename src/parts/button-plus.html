Button+ and others

Buttons on only work on D2L and across all courses as long as D2L does not change its whole awesome iFrame within iFrame structures.

Table of Contents

To go to the table of content, add onclick="gotoToC();" to <a> or <button> element.

Code

<a class="btn default block" href="javascript:void(0)" onclick="gotoToC();">
Table of Contents</a>

Result

Table of Contents

Grades

To go to the grades page, add onclick="gotoGrades();" to <a> or <button> element.

Code

<a class="btn success block" href="javascript:void(0)" onclick="gotoGrades();">Grades</a>

Result

Grades

Dropbox

To go to the dropbox page, add onclick="gotoDropbox();" to <a> or <button> element.

Note: only works when in student mode.

Code

<a class="btn info block" href="javascript:void(0)" onclick="gotoGrades();">Dropbox</a>

Result

Dropbox

Discussions

To go to the discussion page, add onclick="gotoDiscussions();" to <a> or <button> element.

<a class="btn warning block" href="javascript:void(0)" onclick="gotoDiscussions();">
Discussions</a>
