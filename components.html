Components

Tooltip

A short message that appears when the mouse cursor is positioned over an icon, image, hyperlink, or text in a page content.

Tooltip has five options: default, top, bottom, right, and left. Each option will be explained below with code and examples. To use tooltip, add the with-tooltip class and the title attribute to the HTML elements including <span>, <a>, <img>, etc. The title attribute is used to hold the tooltip message. Thus it is required.

NOTE: The <abbr> element does not require the with-tooltip class. When using the <abbr> element, please follow the standards for using an abbreviation.

Default

By default, the tooltip will be displayed on the top of the element without any arrow indication.

Code

<p><abbr title="HyperText Markup Language">HTML</abbr> is the best thing ever.</p>
<!-- can also be done by using the with-tooltip class -->
<p><span class="with-tooltip" title="HyperText Markup Language">HTML</span> is the best
thing ever.</p>

Result

HTML is the best thing ever.

HTML is the best thing ever.

Top

The tooltip will be placed at the top of the text and with an arrow pointing to the element. To specify the top position, add the top class.

Code

<p>The way to go is <abbr class="top" title="PHP: Hypertext Preprocessor">PHP</abbr>.</p>
<!-- can also be done by using the with-tooltip class -->
<p>The way to go is <span class="with-tooltip top" title="PHP: Hypertext Preprocessor">PHP
</span>.</p>

Result

The way to go is PHP.

The way to go is PHP.

Bottom

The tooltip will be placed at the bottom of the text and with an arrow pointing to the element. To specify the bottom position, add the bottom class.

Code

<p>When it is funny, you go <abbr class="bottom" title="laugh out loud">lol</abbr>.</p>
<!-- can also be done by using the with-tooltip class -->
<p>When it is funny, you go <span class="with-tooltip bottom" title="laugh out loud">lol
</span>.</p>

Result

When it is funny, you go lol.

When it is funny, you go lol.

Right

The tooltip will be placed at the right of the text and with an arrow pointing to the element. To specify the right position, add the right class.

Code

<p>The file was not found because the server does not support the correct
<abbr class="right" title="Multipurpose Internet Mail Extensions">MIME</abbr> types.</p>
<!-- can also be done by using the with-tooltip class -->
<p>The file was not found because the server does not support the correct <span
class="with-tooltip right" title="Multipurpose Internet Mail Extensions">MIME</span> types.
</p>

Result

The file was not found because the server does not support the correct MIME types.

The file was not found because the server does not support the correct MIME types.

Left

The tooltip will be placed at the left of the text and with an arrow pointing to the element. To specify the left position, add the left class.

Code

<p>When the server has trouble fulfilling your request, a <abbr class="left"
title="HyperText Transfer Protocol">HTTP</abbr> error code will be displayed instead.</p>
<!-- can also be done by using the with-tooltip class -->
<p>When the server has trouble fulfilling your request, a <span class="with-tooltip left"
title="HyperText Transfer Protocol">HTTP</span> error code will be displayed instead.</p>

Result

When the server has trouble fulfilling your request, a HTTP error code will be display instead.

When the server has trouble fulfilling your request, a HTTP error code will be display instead.

Other Examples:

Tooltip can also be use on hyperlink and image.

Code

<p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Optio, possimus, reprehenderit,
<a class="with-tooltip top" href="javascript:void(0);" title="I have no idea what that
means.">numquam</a> facere corporis necessitatibus atque rerum asperiores quidem maxime
veniam magnam alias enim reiciendis nisi illo eveniet minima quos?</p>

Result

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Optio, possimus, reprehenderit, numquam facere corporis necessitatibus atque rerum asperiores quidem maxime veniam magnam alias enim reiciendis nisi illo eveniet minima quos?

Code

<img class="with-tooltip right" title="Boo! Watch your back." src="src/img/cat1.jpg"
width="316" height="237" alt="Cat" />

Result
