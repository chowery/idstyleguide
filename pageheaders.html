Page Headers

A page must starts with a page header. Page header may not be used multiple times throughout the page.

Simple Page Header

A simple page header contains only the page title.

Code

<header>
	<h1>Page title (30 pixels)</h1>
</header>

Result

Page title (30 pixels)

Page Header with Subtext

A page header with subtext contains the page title with a short supplement text.

Code

<header>
	<h1>Page title (30 pixels) <small>subtext (22 pixels)</small></h1>
</header>

Result

Page title (30 pixels) subtext (22 pixels)
