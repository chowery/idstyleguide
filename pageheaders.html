Page Headers

A person must have a name. A page must starts with a page header.

Regular Page Header

A regular page header contains only the title.

Code

<header>
	<h1>Page title (30 pixels)</h1>
</header>

Result

Page title (30 pixels)

Page Header with Icon

A page header begins with an icon. Please refer to the icons page for a complete list of available icons.

Code

<header>
	<h1><span class="icon icon-discussion"></span>Page title (30 pixels)</h1>
</header>

Result

Page title (30 pixels)

Page Header with Subtext

A page header contains a short supplement text after the title.

Code

<header>
	<h1>Page title (30 pixels) <small>subtext (22 pixels)</small></h1>
</header>

Result

Page title (30 pixels) subtext (22 pixels)
