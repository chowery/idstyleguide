Tables

The default width of the <table> element will always be 100%; thus, it fills up the whole width of the page or container. The <table> is a block-level element. The use of <thead> is optional but recommended. The use of <tbody> is required in table with classes. The cells of the table will auto-resize accordingly to the content they held.

Basic Table

Code

<table>
	<thead>
		<tr>
			<th>ID</th>
			<th>First Name</th>
			<th>Last Name</th>
			<th>Username</th>
		</tr>
	</thead>
	<tbody>
		<tr>
			<td>1</td>
			<td>Ethan</td>
			<td>Lin</td>
			<td>ethan.lin</td>
		</tr>
		<tr>
			<td>2</td>
			<td>Patrick</td>
			<td>Wirth</td>
			<td>patrick.wirth</td>
		</tr>
		<tr>
			<td>3</td>
			<td>Bryan</td>
			<td>Bortz</td>
			<td>bryan.bortz</td>
		</tr>
		<tr>
			<td>4</td>
			<td>John</td>
			<td>Oppenheimer</td>
			<td>john.oppenheimer</td>
		</tr>
		<tr>
			<td>5</td>
			<td>Kristin</td>
			<td>Barnes</td>
			<td>kristin.barnes</td>
		</tr>
		<tr>
			<td>6</td>
			<td>Nate</td>
			<td>Ewings</td>
			<td>nate.ewings</td>
		</tr>
	</tbody>
</table>

Result

2 Patrick Wirth patrick.wirth
3 Bryan Bortz bryan.bortz
4 John Oppenheimer john.oppenheimer
5 Kristin Barnes kristin.barnes

Striped Table

Add "zebra-striping" to odd table rows within the <tbody> element by using the striped class.

Code

<table class="striped"></table>

Cell Alignments

Horizontal Left DEFAULT

Left align the text in the cell by using the horz-left class.

Code

<table class="bordered">
	<tbody>
		<tr>
			<td class="horz-left">Ethan</td>
			<td class="horz-left">lin</td>
			<td class="horz-left">ethan.lin</td>
		</tr>
	</tbody>
</table>

Result

Ethan Lin ethan.lin

Background Colors

			<th>ID</th>
			<th>First Name</th>
			<th>Last Name</th>
			<th>Username</th>

		<tr class="info">
